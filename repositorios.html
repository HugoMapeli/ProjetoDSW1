Manual Git

Passo 2 - Como criar um repositório

Se direcione para a página inicial do GitHub logo após ter feiro login no site.

Passo 2.1

Procure o botão verde "New Repository" e selecione-o.

Passo 2.2

Preencha todos os dados pedidos, como o nome do repositório e a descrição, que não é obrigatória. Para liberar a opção de projeto privadoo, você deve adquirir o plano pago.
