Manual Git

Passo 3 - Git CMD

Passo 3.1

Configure sua identidade no CMD, seguindo o exemplo ao lado.

git config --global user.name "username"

git config --global user.email "email"

Passo 3.2

Para clonar um repositório para seu desktop, você deve ir até a página dele e clicar em "clone ou download" . Copie a url que apareceu e digite no CMD "git clone (url)" , como o exemplo ao lado.

O comando cd é utilizado para navegar entre os locais do computador.

Passo 3.3

Com o comando git status, você pode observar o estado dos seus arquivos ou de um arquivo específico (git status arquivo).

Existem três tipos de estado:

Comitted: quando o arquivo está armazenado no repositório local.

Modified: quando o arquivo foi modificado e ainda não foi salvo no repositório local

Staged: quando o arquivo modificado foi marcado para ser salvo no repositório local no próximo commit.

Passo 3.4

O comando git add prepara um arquivo para commit, adicionando suas modificações.

-Git add arquivo adiciona somente o arquivo selecionado.

-Git add * adiciona todos os arquivos modificados.
